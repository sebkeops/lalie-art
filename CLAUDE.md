# Lalie Art — Contexte projet

## Stack technique
- Next.js 14 (App Router), architecture src/app
- CSS custom via globals.css (pas de Tailwind)
- Supabase (database + storage bucket `artworks`)
- Auth Supabase pour l'espace admin
- Déploiement Vercel

## Design system
Univers sombre, palette CSS :
- `--bg` : fond principal
- `--ink` : texte
- `--wine` : accent principal
- `--rose` : accent secondaire
- `--ivory` : texte clair

Le design est élégant, minimal et artistique.

## Règles de travail
- Respecter l'architecture existante
- Pas de refactoring massif
- Code clair et minimal
- Pas de Tailwind, CSS custom uniquement
- Modifications progressives uniquement
